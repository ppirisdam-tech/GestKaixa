### 1. Objectiu de la Fase

Permetre a l'usuari comprovar l'auditoria i el rastre de totes les operacions fetes en un compte concret. L'historial ha de mostrar de forma cronològica inversa (de més nou a més antic) la data exacta, el concepte de la transacció, l'import extret/ingressat i el saldo resultant després d'aquell moviment.


2.Canvis
Respecte a la v1.1, els canvis principals són:

-Llistat complet d'usuaris des del menú d'administrador, amb comptes associats i saldos.
-Generació automàtica del número de compte (IBAN) en crear un compte nou.
-Registre del usuari_id en cada moviment per a auditoria completa.
-Nova taula Solicituds per gestionar transferències pendents d'aprovació.
-Mètode LlistarComptesDropdown() reutilitzable per evitar repetir codi.



3.Estructura del codi

Continua en un únic fitxer Program.cs,

Main(): Detecció del tipus d'usuari i control de flux principal
IntentarLoginClient(): Login de clients. Ara guarda també el cognom
MenuClient(): Menú de client amb 7 opcions (nova: sol·licitar transferència)
MenuAdministrador(): Menú admin amb 5 opcions
VeureComptesClient(): Llistat de comptes de l'usuari
ConsultarSaldoTotal(): Saldo per compte i total acumulat
VeureMovimentsCompte(): Historial de moviments amb colors
OperarCompte(): Ingrés o retirada. Ara envia usuari_id per a auditoria
VeureAlertesClient(): Alertes del sistema per als comptes de l'usuari
LlistarComptesDropdown(): Nou mètode reutilitzable: llistat de comptes actius de l'usuari, usat per VeureMovimentsCompte, OperarCompte i ClientSollicitarTransferencia
AdminCrearUsuari(): Alta de client amb tots els camps
AdminCrearCompteAutomatic(): Genera un IBAN aleatori i l'insereix
AdminAssignarUsuariACompte(): Vincula usuari i compte amb un rol
AdminLlistarTot(): Nou: llistat complet de tots els usuaris, comptes i saldos
ClientSollicitarTransferencia(): Nou (Fase 2): crea una sol·licitud de transferència pendent
AdminGestionarSolicituds(): Nou (Fase 2): l'admin pot aprovar o denegar sol·licituds
LeerContrasenyaSegura(): Ocultació del password amb asteriscs



4.Explicacio

-A la Fase 2 s'ha afegit el camp usuari_id a la taula Moviments. Ara cada moviment registra qui l'ha fet 

"INSERT INTO Moviments (compte_id, usuari_id, import, concepte) VALUES (@cId, @uId, @imp, @conc)

-El número de compte es genera amb format espanyol: ES + 2 dígits de control + codi d'entitat 2100 + 16 dígits aleatoris, si per casualitat el número ja existeix (UNIQUE a la BBDD), la catch ho detecta i avisa per tornar-ho a intentar

-S'ha creat manualment a MySQL una nova taula per gestionar les transferències:
CREATE TABLE Solicituds (
    id INT AUTO_INCREMENT PRIMARY KEY,
    compte_origen_id INT NOT NULL,
    compte_desti_iban VARCHAR(24) NOT NULL,
    usuari_id INT NOT NULL,
    import DECIMAL(10,2) NOT NULL,
    concepte VARCHAR(120),
    estat ENUM('PENDENT','APROVADA','DENEGADA') DEFAULT 'PENDENT',
    data TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (compte_origen_id) REFERENCES Comptes(id),
    FOREIGN KEY (usuari_id) REFERENCES Usuaris(id)
);
quan un client fa una sol·licitud de transferència, es guarda a Solicituds amb estat PENDENT

    static void ConsultarSaldoTotal(int usuariId)

    {

        Console.Clear();

        Console.WriteLine("          SALDO TOTAL DELS COMPTES      ");

        Console.WriteLine("========================================");

  

        string query = @"

            SELECT c.numero_compte, IFNULL(vs.saldo, 0) AS saldo

            FROM Comptes c

            INNER JOIN UsuarisComptes uc ON c.id = uc.compte_id

            LEFT JOIN VistaSaldos vs ON c.id = vs.compte_id

            WHERE uc.usuari_id = @usuariId";

  

        decimal saldoTotalAcumulat = 0;

  

        using (MySqlConnection conn = new MySqlConnection(connectionString))

        {

            try

            {

                conn.Open();

                using (MySqlCommand cmd = new MySqlCommand(query, conn))

                {

                    cmd.Parameters.AddWithValue("@usuariId", usuariId);

  

                    using (MySqlDataReader reader = cmd.ExecuteReader())

                    {

                        if (!reader.HasRows)

                        {

                            Console.WriteLine("[!] No tens cap compte per consultar el saldo.");

                        }

                        {

                            Console.WriteLine("----------------------------------------");

                            while (reader.Read())

                            {

                                decimal saldoCompte = Convert.ToDecimal(reader["saldo"]);

                                saldoTotalAcumulat += saldoCompte;

  

                                Console.WriteLine("{0,-25} {1,14:C2}",

                                    reader["numero_compte"],

                                    saldoCompte);

                            }

  

                            Console.WriteLine("----------------------------------------");

                            Console.ForegroundColor = ConsoleColor.Green;

            {

                Console.WriteLine($"[!] Error en consultar el saldo: {ex.Message}");

            }

        }

    static string LeerContrasenyaSegura()

    {

        string pass = "";

        ConsoleKeyInfo key;

        do

        {

            key = Console.ReadKey(true);

            if (key.Key != ConsoleKey.Backspace && key.Key != ConsoleKey.Enter)

            {

                pass += key.KeyChar;

                Console.Write("*");

            }

            else if (key.Key == ConsoleKey.Backspace && pass.Length > 0)

            {

                pass = pass.Substring(0, (pass.Length - 1));

                Console.Write("\b \b");

            }

        } while (key.Key != ConsoleKey.Enter);

        Console.WriteLine();

        return pass;

    }
